Set up AD, OUs, and add users from CSV

This script creates 21 organizational units in an Active Directory environment, then pulls dummy csv files (fake names) with the headers 'first_name' and 'last_name' off of my PowerShell_Scripts GitHub repository. It will then create a default password for each user that will consist of the domain name and the enrollment date. As OUs and Users are being created, there will be a message stating that the OU or User has been created and added to Active Directory. This script expedites the Active Directory setup process by parsing and adding users via CSV files and PowerShell, instead of manually. This script can be adjusted based on your organization's needs. I highly recommend creating your own secure algorithm for password creation and to store your csv files in a secure place. This script can be used, but should be modified accordingly.

Enable or disable inactive users (GUI/.Net)

This script will find users in Active Directory who have NEVER logged in and then allow the admin to either disable an individual user, or disable all the users who have never logged in for a given OU. The admin can also enable the users that were disabled as well. This script has a GUI that was created with WinForms (.NET).
