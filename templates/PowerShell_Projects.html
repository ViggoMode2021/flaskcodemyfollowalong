Set up AD, OUs, and add users from CSV

This script creates 21 organizational units in an Active Directory environment, then pulls dummy csv files (fake names) with the headers 'first_name' and 'last_name' off of my PowerShell_Scripts GitHub repository. It will then create a default password for each user that will consist of the domain name and the enrollment date. As OUs and Users are being created, there will be a message stating that the OU or User has been created and added to Active Directory. This script expedites the Active Directory setup process by parsing and adding users via CSV files and PowerShell, instead of manually. This script can be adjusted based on your organization's needs. I highly recommend creating your own secure algorithm for password creation and to store your csv files in a secure place. This script can be used, but should be modified accordingly.

Add student or staff account to AD

With this script, the admin can create either a student or staff account and have them added to the proper organizational unit, as well as creating them a default password, setting their home directory, and mapping their network drive.

Move OUs at end of school year

In a school themed (K-12) Active Directory environment, the graduating class' child OU of the parent 'High_School_Top_OU ' OU will get archived (moved to an OU named 'Graduated_Classes', and will create this OU if it doesn't exist), the 8th grade OU will get moved to the High-School OU, the 4th grade child OU of the 'Elementary_School_Top_OU' will be moved to the Middle-School OU, and a new OU for the incoming elementary school class. Before the script moves these OUs accordingly, it makes sure that this script is run AFTER July 1st of the current year, in order to assure that the school year has properly ended and changes to the AD environment can be made. It also will dynamically, create the incoming OU's graduating class by adding 12 to the current year. For example, the incoming class to a new school will be titled 'Class_Of_2035' if the script is run in 2023 after July 1st.

Enable or disable inactive users (GUI/.Net)

This script will find users in Active Directory who have NEVER logged in and then allow the admin to either disable an individual user, or disable all the users who have never logged in for a given OU. The admin can also enable the users that were disabled as well. This script has a GUI that was created with WinForms (.NET).

Active Directory Password Generator

This script will generate a password for a user based on a csv file that they have. The admin will select a csv file on their local computer, select a user, and then click the 'Generate Password' button. It will display the password for the new user in plaintext and allow it to be copied to the clipboard. If the password does not meet the complexity of the domain, the script will append numbers and/or special characters in order to create an adequate password. This script also has a GUI that was created with WinForms (.NET).

Timesheet Automator

This script automates and replicates the process of creating a new timesheet every Monday and logging the user's clock in time when the script is run. It automatically calculates the hours worked by subtracting 30 minutes (lunch break) from the difference between the clock in time to the standard clock out time of 5:00 PM. The script copies a standard template of a timesheet (Word document) at the start of the week and names it "TimeSheet-Week-Of-7-17-2023" or whatever date. It runs the same way for Tuesday-Thursday. On Friday, it calculates the week's total hours worked by adding up the number of hours worked each day. It then prints out the timesheet.

Network-Necromancer

Network Necromancer is a PowerShell script that automates adding, removing, and connecting to networks by implementing variations of 'netsh wlan show profiles' in the backend. This script will easily clean up networks stored on a computer's NIC, export and import network based XML files, and more. For example, a user can utilize this script to move the networks stored on one computer onto another computer without manually connecting to networks and inputting passwords.

Partition Pantheon

Partition-Pantheon is a flash drive/harddrive partitioning automation script that will seamlessly automate the process of formatting, naming, setting letters, and allocating size to partitions. It will also move files to partitions based on file extensions.
